# decks — эталонные структуры презентаций эфиров

**Статус на 2026-06-04: эталон есть.** Markdown-структура слайдов в формате `methodist` (заголовок → тип → что на слайде → что говорит спикер → переход) собрана.

## Эталоны

- `2026-06_openclaw-praktikum_deck.md` — **продающий вебинар-практикум** «Новый практикум по OpenClaw» (62 слайда). Импульсная дуга: крючок → доверие → ядро → демо → оффер → отработка. Для тем с продажей на эфире (оплати/успей).
- `2026-06_ii-consulting-webinar_deck.md` — **образовательно-консультативный вебинар** (ИИ-консалтинг, ~140 слайдов). Дуга: глубокая польза → проблематизация на данных → доказательство (кейсы + внутренняя кухня) → мягкий оффер через заявку на собеседование. Для премиальных/сложных продуктов, где решают через диалог. Две «двери»-оффера под разные сегменты + ритм вовлечения в чат.

Два эталона = два регистра продающего эфира: импульсный (демо+дедлайн) и экспертный (польза+заявка). Под запуск выбираем по типу продукта и решения.
