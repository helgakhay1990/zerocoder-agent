# Презентация: Новый практикум по OpenClaw

## Общая информация

- **Длительность:** ~1,5–2 часа (заявлено на слайде 4)
- **Спикер:** Кирилл Пшинник, CEO и co-founder Университета Zerocoder
- **Главное сообщение:** OpenClaw — это не «ответ на сегодня» (как Claude Code), а собранный сотрудник, который работает за вас постоянно, по расписанию и без вас.
- **Желаемое действие зрителя:** купить «Практический курс. Вайб-кодинг с OpenClaw» прямо в эфире по промокоду (скидка + бонусы за скорость).
- **Объём:** 62 слайда.

## Ритм-карта

Дуга — каноничный продающий вебинар: **разогрев → доверие к спикеру → вовлечение → проблема/хайп → содержательное ядро → демо-кейс → оффер → доверие к компании → программа и результаты → тарифы → отработка возражений → бонусы за скорость → Q&A**.

Ключевой ритм-приём: содержательные блоки **никогда не идут длинной простынёй** — каждые 3–6 слайдов вставляется либо интерактив («напишите в чат N», «ставьте 1»), либо повтор оффера с дедлайном «пока идёт эфир». Это держит вовлечение и трижды прокатывает оффер мимо зрителя до финального тарифного блока.

## Слайды (образцовые, в формате методиста)

Ниже — не все 62, а опорные слайды каждого типа: на них видно глубину проработки, которую держит `methodist`. Остальные собираются по той же логике в рамках своего блока.

### Слайд 4. План на ближайшие 1,5–2 часа
- **Тип:** тезис (контракт со зрителем)
- **Что на слайде:** обещание разобрать разницу OpenClaw vs Claude Code + сквозной кейс (агент обработки заявок) по 4 вопросам: каркас → настройка → отчёты/управление → усиление.
- **Что говорит спикер:** проговаривает маршрут эфира, прямо называет, ради чего стоит досидеть до конца.
- **Длительность:** 2–3 мин.
- **Переход:** «и за то, что досидите, будут подарки» → слайд 5–6.

### Слайд 17. OpenClaw — это сборка сотрудника, который остаётся
- **Тип:** тезис (ядро смысла)
- **Что на слайде:** короткие контрастные пункты — работает без вас, по расписанию/событию; результат не файл, а процесс; работает, когда вы не пишете; остаётся, даже если вы в отпуске.
- **Что говорит спикер:** раскрывает метафору «собрать работника, а не получить ответ»; на контрасте с предыдущим слайдом про Claude Code.
- **Длительность:** 3–4 мин.
- **Переход:** к слайду 18 — одна фраза-якорь, сжимающая всю разницу в две строки.

### Слайд 19. Оффер «пока идёт эфир» (ранний прокат)
- **Тип:** оффер
- **Что на слайде:** 🔥 скидка 5000 ₽ по промокоду OPENCLAW5 + виртуальная карта с $20 + прокси на период обучения. Дисклеймер про прокси.
- **Что говорит спикер:** «не дожидаясь конца — для самых быстрых»; создаёт первую точку срочности задолго до тарифов.
- **Длительность:** 1–2 мин.
- **Переход:** «а теперь покажу, как это работает на деле» → демо-кейс (20–24).

### Слайд 21. Как собрать каркас агента
- **Тип:** демо (пошаговое)
- **Что на слайде:** 5 шагов — регистрация и New Agent → имя/язык/модель → промпт → источники данных → тест на 5–10 заявках.
- **Что говорит спикер:** показывает живьём (демонстрация интерфейса), проговаривает каждый шаг. ⚠️ Зона демо-инструментов — на монтаже проверять даты на экране (см. чек-лист монтажа).
- **Длительность:** 5–7 мин.
- **Переход:** «каркас есть — теперь научим обрабатывать качественно» → слайд 22.

### Слайд 28. «Кто захотел научиться? ставьте 1»
- **Тип:** интерактив
- **Что на слайде:** один вопрос крупно + «Ставьте 1».
- **Что говорит спикер:** замеряет тёплых, поднимает активность чата перед продуктовым блоком.
- **Длительность:** <1 мин.
- **Переход:** парный интерактив (слайд 29 «ставьте 2») → продукт (30).

### Слайд 49. Цены по тарифам
- **Тип:** оффер (ядро продажи)
- **Что на слайде:** три тарифа с перечёркнутой и актуальной ценой: Базовый 75 000 → 29 900, Бизнес 85 000 → 39 900, ВИП 150 000 → 59 900.
- **Что говорит спикер:** проговаривает ценность каждого, ведёт к среднему (Бизнес) как рекомендуемому.
- **Длительность:** 3–5 мин.
- **Переход:** «а для самых быстрых — ещё скидка» → доп. промокод (50–53).

### Слайд 55. Гарантия возврата
- **Тип:** тезис (снятие риска)
- **Что на слайде:** «Попробуйте: если в течение 14 дней поймёте, что не подходит — вернём всю сумму».
- **Что говорит спикер:** снимает финансовый страх перед решением, обнуляет риск покупки.
- **Длительность:** 1 мин.
- **Переход:** к бонусам за скорость оплаты (56–57).

## Паттерны, которые стоит переиспользовать

- **Крючок-якорь в начале и закрытие в конце.** Подарки за досмотр анонсируются на слайдах 5–6, выдаются на 59–60 — рамка, держащая зрителя весь эфир.
- **Тройной прокат оффера.** Оффер показывается трижды (19 → 25–26 → 56–57) с нарастающей срочностью, а не один раз в финале.
- **Содержание дробится интерактивом.** Между смысловыми блоками — «напишите в чат N» / «ставьте 1». Ни один блок не идёт простынёй.
- **Контрастная пара слайдов под главную мысль.** 16 (Claude Code) ↔ 17 (OpenClaw) ↔ 18 (фраза-якорь) — сначала развёртка контраста, потом сжатие в одну строку.
- **Блок доверия — единым массивом перед ценой.** Метрики, лицензия, резиденты, клиенты, вузы, госслужащие, СМИ (31–39) идут плотно перед тарифами, чтобы снять «кто вы такие» до разговора о деньгах.
- **Демо как доказательство, а не как лекция.** Кейс один сквозной (агент заявок), разобран по 4 заранее обещанным вопросам из плана — план и демо зеркалят друг друга.
