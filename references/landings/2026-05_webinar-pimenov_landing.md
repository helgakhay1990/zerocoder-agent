# Структура лендинга: вебинар с приглашённым экспертом + апсейл-тариф

Логика повествования: **захват (кто + что + бонус) → доказательство ценности → программа → авторитет спикеров → выбор участия → отстройка «для кого» → доверие к школе → финальный призыв → снятие барьера (глоссарий)**.

## Блок 1. Hero (первый экран)
- Фото спикера + плашка с именем приглашённого эксперта (**Сергей Пименов**) и ведущего (**Кирилл Пшинник**).
- **H1:** «Бизнес-разбор с ИИ-консультантом» — суть формата сразу.
- Подзаголовок-вопрос, бьющий в выгоду: «Как зарабатывать на ИИ-консалтинге с командой из ИИ-агентов?»
- **CTA:** «Зарегистрироваться».
- **Бонус-крючок прямо на первом экране:** «Всем участникам подарим чек-лист "10 точек автоматизации в любом бизнесе"».
- *Логика:* за один экран — кто, что, зачем и что получишь бесплатно.

## Блок 2. Что покажет эксперт (light)
- **H2:** «Сергей Пименов покажет в деталях».
- Раскрытие ценности эфира до программы — почему стоит остаться.

## Блок 3. Программа эфира (dark, #program)
- **H2:** «Что будет в эфире?» — список того, что разберут.
- **CTA по ходу:** «Я с вами».
- *Логика:* тёмная секция визуально отбивает программу как смысловой центр.

## Блок 4. Спикеры (light, #speakers)
- **H2:** «Спикеры» → карточки: **Кирилл Пшинник** (ведущий, ген. директор) и **Сергей Пименов** (приглашённый эксперт).
- Регалии каждого — авторитет до оффера.

## Блок 5. Варианты участия / тарифы (light, #tariffs)
- **H2:** «Варианты участия». Два тарифа:
  - **Бесплатный:** участие в практикуме, возможность задать вопросы онлайн. CTA «Зарегистрироваться бесплатно».
  - **Бизнес (499 ₽):** всё из бесплатного + стратегическая сессия с экспертом Университета (обычно 10 000 ₽) + запись эфира + доступ в НЕЙРОКЛУБ на 1 месяц. CTA «Зарегистрироваться за 499 ₽».
- *Логика:* бесплатный вход снимает барьер, платный апсейл с явной экономией (10 000 → 499) монетизирует тёплых.

## Блок 6. Для кого этот эфир (light, #audience)
- **H2:** «Для кого этот эфир» → 5 сегментов карточками: эксперты-практики; руководители и топ-менеджеры; предприниматели и собственники МСБ; маркетологи/продюсеры/продакты/аналитики; те, кто пробовал нейросети, но боится технической стороны.
- *Логика:* читатель узнаёт себя → «это для меня».

## Блок 7. О школе (soft, #about)
- **H2:** «Мы создаём фундаментальное образование в области ИИ и разработки».
- Подблок доверия: «Проводим исследования по ИИ совместно с лучшими вузами страны».
- *Логика:* институциональное доверие перед финальным призывом.

## Блок 8. Финальный CTA (dark, .final-cta)
- **H2:** дублирует название разбора. CTA «Посетить разбор».
- *Логика:* повтор оффера тёмной секцией на выходе — последняя точка конверсии.

## Блок 9. Глоссарий терминов (light, #glossary)
- **H2:** «Глоссарий терминов».
- *Логика:* снимает барьер у новичков, боящихся «технической стороны» (перекликается с сегментом из блока 6).

## Паттерны, которые стоит копировать

- **Бонус-крючок на первом экране** (чек-лист в подарок) — усиливает первый CTA.
- **Два тарифа с явной экономией** на платном (10 000 → 499 ₽) — бесплатный снимает барьер, платный монетизирует.
- **CTA повторяется** на каждом смысловом рубеже (hero → программа → тарифы → финал), формулировки разные («Я с вами», «Принять участие», «Посетить разбор»), не одинаковые кнопки подряд.
- **Глоссарий в конце** — нестандартный, но рабочий ход под сегмент «боюсь технической стороны».
- **Ритм секций** light/dark/soft — тёмные секции отбивают смысловые центры (программа, финальный CTA).
