# Чек-лист качества: анонс

> Использует `announce-writer` перед отдачей черновика Повелительнице. Если хотя бы один пункт из «обязательных» проваливается — переписываем, не отдаём.
> Опирается на [tone-of-voice.md](../knowledge-base/tone-of-voice.md) и [audience.md](../knowledge-base/audience.md).

---

## Обязательные (без них не отдавать)

- [ ] **Крючок в первом абзаце** — боль или интригующий факт, не «здравствуйте / приглашаем / рады представить».
- [ ] **Конкретная польза** — что человек узнает или унесёт. Не «много нового и интересного», а конкретные пункты.
- [ ] **Дата, время, формат** — указаны явно.
- [ ] **Спикер** — назван по имени (для эфиров Кирилл Пшинник, для остальных — соответствующий эксперт).
- [ ] **CTA** — конкретный, с действием и ссылкой/кнопкой. Не «оставьте заявку».
- [ ] **Нет канцелярита** из стоп-списка (осуществлять, в рамках, данный, является, производить, мероприятие).
- [ ] **Нет англицизмов** из стоп-списка (митинг, таски, фидбэк, апрув, лонгрид).
- [ ] **Нет инфо-цыганских клише** («уникальная возможность», «не упустите шанс», «эксклюзивный», «изменит вашу жизнь», «секреты»).
- [ ] **Тон по сегменту** — если анонс под конкретный сегмент (спецы / предприниматели / руководители / эксперты / энтузиасты), язык и фокус соответствуют этому сегменту.

## Желательные

- [ ] **Использована фраза из языка ЦА** (см. [audience.md](../knowledge-base/audience.md) по соответствующему сегменту).
- [ ] **Закрыто одно возражение** этого сегмента в теле текста (например, «не для гиков», «без VPN», «с гарантией возврата»).
- [ ] **Прехедер не повторяет тему** письма (для email).
- [ ] **Эмодзи** — максимум 2–3 на единицу текста, каждый работает на ритм, не для украшения.
- [ ] **Длина предложений** — короткие и средние, без перегруза.
- [ ] **Промокод или киллер-предложение** упомянуты в финале (если оффер с дедлайном).

## Стоп-сигналы (если есть хотя бы один — переписать)

- [ ] Начало «Здравствуйте», «Приглашаем», «Уважаемые», «Дорогие друзья», «Рады представить», «С радостью сообщаем».
- [ ] Эмодзи-салют в начале (🎉🚀✨🔥) — инфо-цыганский маркер.
- [ ] Восклицания в каждом предложении.
- [ ] Длинная подводка про компанию до пользы для читателя.
- [ ] Гипертрофированные обещания: «изменит всё», «навсегда», «революция», «100% результат», «успех гарантирован».
- [ ] «Все мы знаем, что…» — нет, не знаем.
- [ ] Утверждения без подтверждения («наши студенты зарабатывают в 3 раза больше»).
- [ ] Выдуманные продукты, цены, тарифы — всё из [zerocoder-facts.md](../knowledge-base/zerocoder-facts.md).

## Формат самопроверки

После написания черновика — пройти по списку сверху вниз. Если попадаешь в стоп-сигнал — переписать раздел и пройти заново. Если 2+ желательных не закрыты — подумать, можно ли усилить, не утяжелив.

Финальная проверка вслух: прочитать текст «как читатель из ЦА». Если на второй фразе хочется закрыть — переписывать.
