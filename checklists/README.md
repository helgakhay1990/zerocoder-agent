# checklists — проверочные листы качества

Каждый контентный субагент перед отдачей работы Повелительнице прогоняет результат через свой чек-лист. Это резко поднимает планку, потому что агент не отдаёт первый черновик — фильтрует себя сам.

## Файлы (создаются постепенно)

- **`announce-quality.md`** — для `announce-writer`. Есть ли крючок в первом абзаце, есть ли явная польза, есть ли срочность, есть ли CTA, нет ли клише, попали ли в ToV.
- **`landing-quality.md`** — для `landing-architect`. Соответствует ли референсу, есть ли логика прокрутки, нет ли смысловых дыр между блоками, дублируются ли CTA, нет ли воды.
- **`concept-quality.md`** — для `launch-strategist`. Не выдуманы ли продукты, проверены ли цифры, прописана ли ЦА конкретно, есть ли KPI, есть ли риски.
- **`deck-quality.md`** — для `methodist` (презентации). Ритм слайдов, нет ли «слайдов-простыней», есть ли визуальные паузы, длительность общая.
- **`guide-quality.md`** — для `methodist` (гайды). Пошагово, без воды, каждый шаг даёт обучаемому конкретный результат.

## Формат чек-листа

Простой markdown с пунктами «да/нет» + критерии прохождения. Пример:

```markdown
# Чек-лист качества: анонс

## Обязательные (без них не отдавать)
- [ ] Есть крючок в первом абзаце (не «приглашаем»)
- [ ] Есть конкретная польза (не «много нового и интересного»)
- [ ] Есть конкретный CTA с датой/ссылкой
- [ ] Нет канцелярита и англицизмов из списка анти-паттернов

## Желательные
- [ ] Использована фраза из языка сегмента ЦА
- [ ] Прехедер не повторяет тему
- [ ] Эмодзи — максимум 2-3, и каждый работает на ритм

## Стоп-сигналы (если есть хотя бы один — переписать)
- [ ] Начало «Здравствуйте», «Приглашаем», «Уважаемые»
- [ ] Клише: «уникальная возможность», «не упустите шанс»
- [ ] Длинная подводка про компанию до пользы
```

## Что НЕ кладём

- Не оформляем чек-листы как длинные методические тексты — это рабочий инструмент, должно быть быстро.
- Не дублируем здесь CLAUDE.md и общие правила.
- Не делаем чек-листы «на всякий случай» — только под реальные жанры, которые регулярно идут.
