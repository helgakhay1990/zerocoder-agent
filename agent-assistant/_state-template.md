## Этапы конвейера

Отметить нужные под формат (см. «Конвейерные сценарии» в `CLAUDE.md`). Лишние — вычеркнуть.

### Вебинар
- [ ] **Концепция** — `launch-strategist` (внутри: `audience-researcher` + `offer-mechanic`) → `*_concept.md`
- [ ] **Структура лендинга** — `landing-architect` → `*_landing.md`
- [ ] **Анонсы по волнам** — `announce-writer` ×N: email-1, email-2, tg-канал, tg-бот
- [ ] **Презентация эфира** — `methodist` → `*_deck.md`
- [ ] **Гайд участника** — `methodist` → `*_guide.md`
- [ ] **Готовность эфира** — скилл `Webinar-Prep` → `*_prep.md`
- [ ] **Финальная вычитка** всех текстов — `editor`
- [ ] *(после эфира)* **ТЗ на монтаж** — `video-edit-assistant` → отдельный `_state.md` монтажной ветки

### Акция к дате
- [ ] **Разведка ниши** — `analyst-assistant` (что у конкурентов в эту дату/тему)
- [ ] **Концепция акции** — `launch-strategist` (опирается на аналитику)
- [ ] **Страница акции** — `landing-architect`
- [ ] **Анонсы по волнам** — `announce-writer` ×N
- [ ] **Финальная вычитка** — `editor`

### Интенсив
- [ ] **Аналитика рынка** — `analyst-assistant`
- [ ] **Концепция** — `launch-strategist`
- [ ] **Лендинг** — `landing-architect`
- [ ] **Анонсы прогрева** — `announce-writer` (серия касаний)
- [ ] **Гайд участника** — `methodist`
- [ ] **Презентации модулей** — `methodist` (по модулю на этап)
