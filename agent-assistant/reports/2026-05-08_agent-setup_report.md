## Структура проекта

```
agent-assistant/
├── .claude/

    └── 2026-05-08_agent-setup_report.md
```

## Где лежат настройки

| Файл | Назначение |
|---|---|
| `CLAUDE.md` | Роль агента, правила поиска, безопасности и нейминга файлов |
| `.mcp.json` | Конфигурация MCP-серверов: Brave Search (веб-поиск) и Fetch (чтение страниц) |
| `.claude/settings.local.json` | Локальные разрешения: автоматический запуск MCP-инструментов без подтверждения |

## Подключённые MCP-серверы

- **brave-search** — веб-поиск через Brave Search API (ключ настроен)
- **fetch** — загрузка и чтение веб-страниц (без ключа)

## Задачи агента

- Конкурентный анализ
- Мониторинг новостей IT и AI
- Разбор вебинарных видео на ошибки и спорные моменты
- Правка презентаций при изменении тарифов
- Составление ТЗ на правки сайтов и материалов
- Сборка данных для лид-магнитов
