## Что ещё бывает на монтажах (чек-лист, добавить раздел при необходимости)

Базовые три раздела (поправить / вырезать / заблюрить) закрывают большинство случаев, но не всё. Под каждый эфир пройти этот список и **добавить отдельный раздел** в ТЗ, если задача актуальна:

- **Заменить слайд с устаревшим оффером** — таймкоды + ссылка на актуальный слайд («ставим вместо старого»).
- **Перебить речь озвучкой** — например, старая цена/гарантия в речи спикера → таймкод + текст новой озвучки.
- **Вырезать целый блок** — устаревшая акция, снятый с продаж сервис (бесплатный Perplexity, ВПН-гайд, старый состав нейроабонемента). Диапазоны вырезания + пометка «склейка нужна».
- **Добавить новый слайд / вставку** — например, гарантия возврата, новый бонус, обновлённый бандл. Указать, между какими таймкодами вставлять и какой материал.
- **Звуковая обработка** — шумодав, балансировка громкости между спикерами, удаление эха.
- **Интро / аутро** — обновить плашки в начале и конце.
- **Нарезка коротких клипов** — отдельный список фрагментов для соцсетей (3–7 шт по 30–90 сек), каждый с таймкодом и темой.
- **Субтитры / расшифровка** — нужны ли SRT-субтитры в видеоплеере или отдельная расшифровка эфира текстом.

При наличии любого из этих пунктов — создавать в ТЗ свой раздел с понятным заголовком и таблицей таймкодов.

## Перед отдачей — финальная сверка с Ольгой

После заполнения ТЗ:

1. Прислать Ольге готовый файл и **дождаться сверки** — она может найти пропущенные слайды, фрагменты или попросить переформулировать.
2. Внести правки.
3. Только после её подтверждения — копировать содержимое в Google Doc / отдавать монтажёру.

**Не отправлять ТЗ во внешние каналы (Slack, монтажёру, в общий Google Doc) от имени Ольги или самостоятельно — только готовый черновик для её проверки.**
