💡 Идеи по автоматизации работы юристов

Концепция использования Code-Code (нейросети)

Идея 1: Деперсонализация документов 10:04

  • Использовать Code-Code для быстрого создания кода, проверяющего договоры
  • Преимущество: документы остаются локально на компьютере, не передаются в языковую модель
  • Процесс:
  • Берешь договор
  • Скармливаешь его Code-Code
  • Просишь написать код, который ищет персональные данные и заменяет их на вымышленные
  • Создается словарь соответствий (например: "Дарья Копылова" → "Вася Пупкин")
  • Можно обратно раскатать договор с реальными данными

Идея 2: Юридический аудит сайтов 10:58

  • Создать скилл, который делает аудит сайта на юридическую безопасность
  • Проверяет:
  • Пользовательские соглашения и их классификацию
  • Соответствие галочек и элементов законодательству
  • Открытые данные (пользовательские соглашения доступны публично)

Идея 3: Аудит контрагентов 11:38

  • Теоретически можно делать юридический аудит контрагентов, ища данные в открытых источниках
  • Требует дополнительной проверки возможности реализации

Рыночная возможность 💰

09:11 Валерий выделил интересную тему:

  • Проблема: Роскомнадзор требует документы по трансграничной передаче данных, оператор персональных данных, англосицизмы
