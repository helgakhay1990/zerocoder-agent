# Montage-TZ — запуск с нуля (для коллег)

Скилл собирает **черновик ТЗ на монтаж** записи эфира: транскрипт по всей записи (whisper) + OCR по окнам шеринга экрана (tesseract) → размеченный список таймкодов на проверку.

Логика и границы — в `SKILL.md`. Здесь — как поставить и запустить.

---

## Что нужно поставить

| Инструмент | Зачем | macOS (Homebrew) |
|---|---|---|
| `ffmpeg` | извлечение аудио и кадров | `brew install ffmpeg` |
| `whisper-cli` (whisper.cpp) | транскрипт | `brew install whisper-cpp` |
| `tesseract` | OCR кадров | `brew install tesseract` |
| Python 3.9+ | сам скрипт | системный/`brew install python` |

Скрипт работает на стандартной библиотеке Python — доп. пакеты не нужны. При первом запуске сам докачает модель whisper в `../../models/`.

### OCR русского текста (опционально)

По умолчанию OCR на `eng` — этого хватает для дат, латиницы и терминала (`Last login`, `2026-06-03`, `sk-...`). Для распознавания **русского текста в интерфейсе** доставь языковые данные:

1. Узнай папку tessdata: `tesseract --list-langs` (в первой строке путь).
2. Скачай `rus.traineddata` (github.com/tesseract-ocr/tessdata) в эту папку.
3. Запускай с `--ocr-lang rus` (или `--ocr-lang rus+eng`).

## Запуск

```bash
# из корня проекта
python3 agent-assistant/skills/Montage-TZ/montage_tz.py \
    --source <kinescope_id|путь_к_видео> \
    --windows "20m-50m,1h15m-1h40m" \
    --theme webinar-tema
```

**Окна шеринга экрана** (`--windows`) — интервалы, где на экране показывали терминал/браузер/таблицы. Спроси у автора эфира, он знает сценарий. Формат: `20m-50m`, `1h15m30s-1h40m`, либо `MM:SS`/`HH:MM:SS`. Без окон соберётся только речевой слой.

### Источник записи

- **Локальный файл:** `--source путь/к/видео.mp4`.
- **Kinescope по id:** задай токен в окружении — `export KINESCOPE_API_TOKEN=...` — и `--source <video_id>`. Токен у каждого свой (см. `SETUP.md`), в скрипт не зашит.

### Полезные флаги

- `--whisper-model base` (дефолт) — быстро; `small`/`medium` на мощной машине для качества.
- `--quality 480p` (дефолт) — качество скачивания из Kinescope.
- `--ocr-step 5`, `--frame-step 30` — плотность скана внутри окон (сек).
- `--out путь.md` — куда положить черновик (по умолчанию `reports/`).
- `--refresh` — пересобрать, игнорируя кэш.

Полный список: `python3 montage_tz.py --help`.

## Что на выходе

`reports/[YYYY-MM]_[тема]_montazh-tz-draft.md` — **черновик-кандидаты** по разделам:
заблюрить/заменить на экране · вырезать из речи · даты в речи · чистка речи · карта структуры.

Это не финал: дальше — проверка через `video-edit-assistant` (отсев OCR-шума, решения «резать/блюрить») и сверка с автором. Только потом ТЗ идёт монтажёру.

## Если что-то не так

- **«Не найдены инструменты…»** — поставь недостающее (таблица выше).
- **Пустой экранный слой** — не заданы `--windows`; добавь окна шеринга.
- **Русский текст на экране не распознаётся** — доставь `rus.traineddata` (см. выше).
- **Kinescope: «не файл / нет токена»** — задай `KINESCOPE_API_TOKEN` или передай локальный путь.
- **whisper долго на длинной записи** — это нормально (модель `base` идёт фоном); на слабой машине не поднимай модель выше `base`.
