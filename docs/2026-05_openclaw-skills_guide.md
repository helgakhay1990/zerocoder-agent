## Что такое навыки OpenClaw

Навыки (skills) — это подключаемые модули, расширяющие возможности вашего локального AI-ассистента OpenClaw.

С помощью навыков OpenClaw может выполнять действия за пределами обычного чата: обращаться к внешним сервисам, управлять файлами, автоматизировать задачи и многое другое. Навыки оформляются по стандарту Agent Skills (разработан Anthropic) и представляют собой описание действий, которые агент должен выполнять в ответ на определённые запросы пользователя.

OpenClaw поставляется с набором официальных навыков (около 49 штук), покрывающих наиболее распространённые сценарии. Эти встроенные навыки охватывают экосистемы Apple, Google, коммуникации, умный дом, инструменты разработчика и т. д. Кроме того, сообщество создало тысячи сторонних навыков, опубликованных в открытом репозитории (ClawHub). Ниже составлен подробный каталог основных навыков — как официальных, так и интересных сторонних — с описанием их назначения, требований для работы и полезности для пользователя.

> **Замечание:** Перед установкой навыка убедитесь, что он безопасен. OpenClaw сотрудничает с VirusTotal для проверки навыков на угрозы. Рекомендуется просматривать исходный код навыков или отчёты VirusTotal на ClawHub, чтобы избежать вредоносных расширений.

## Установка и управление навыками

Установка навыков выполняется либо через командную строку, либо прямо через ассистента.

**Через ClawHub CLI** — самый простой способ: выполните команду в терминале:
```
npx clawhub@latest install <slug-навыка>
```
Можно также скопировать папку навыка в директорию `~/.openclaw/skills/` (глобально) или в `./skills/` внутри проекта.

**Через чат** — попросите самого OpenClaw установить навык, отправив в чат ссылку на репозиторий навыка. Например: «Установи навык X» — при наличии навыка ClawHub ассистент сам скачает и подключит его.

> **Важно:** Многие навыки имеют внешние зависимости — требуются учётные данные (API-ключи, токены), установленные утилиты или библиотеки. Если зависимость не удовлетворена, навык не будет активен, пока вы не настроите окружение.

## Рекомендованные навыки: с чего начать

Эти навыки подойдут практически каждому пользователю. Они помогут ассистенту искать актуальную информацию, управлять почтой и календарём, получать сводки погоды, работать с документами и облегчать повседневные задачи. Особенно ценны для пользователей 40+, не обладающих глубокими техническими знаниями, которые только установили OpenClaw и хотят быстро получить пользу от ассистента.

### Поиск информации в интернете

**Bing Search** — навык для веб-поиска через поисковик Bing. Позволяет задавать ассистенту вопросы о свежей информации, новостях, фактах — агент выполнит поисковый запрос и обработает результаты.
- **Требования:** возможно, требуется API-ключ Bing (Azure Cognitive Services). Навык помечен как доступный «для всех пользователей», что подразумевает простую настройку.
- **Полезность:** даёт ассистенту доступ к актуальным данным в интернете в режиме реального времени, чего нет у базовой языковой модели.

**DuckDuckGo Search** — навык поиска через DuckDuckGo. Альтернатива Бингу, не требующая ключа API.
- **Требования:** отсутствуют (используются открытые возможности DDG или встроенные API без ключа).
- **Полезность:** позволяет получить результаты веб-поиска без сложной настройки. DuckDuckGo славится конфиденциальностью. Ассистент сможет найти информацию, ответить на вопросы о последних событиях, продуктах, сайтах и т. п.

> **Примечание:** Существуют и другие навыки для поиска — например, Google Search (через Custom Search API), Brave Search и др. — но они потребуют регистрации API-ключей. Для начала Bing или DDG проще всего.

### Электронная почта и календарь

**Gmail** — навык для работы с электронной почтой Gmail. Позволяет читать входящие письма, отвечать на них, отправлять новые письма от вашего имени и выполнять поиск по почтовому ящику.
- **Требования:** Google API credentials (необходимо подключить аккаунт Gmail через OAuth). Настройка включает получение токена OAuth2 — нужно создать проект в Google Cloud Console, включить Gmail API и предоставить доступ ассистенту.
- **Полезность:** очень высока для занятых пользователей — вы можете попросить ассистента «прочитать новые письма» или «ответить на письмо от Анны», и навык выполнит это автоматически. Для пользователей 40+ это снимает рутину работы с почтой, экономя время.

**Outlook** — навык для работы с почтой Microsoft Outlook/Office 365. Позволяет просматривать и искать письма, отправлять письма, а также управлять календарём Outlook через Microsoft Graph API.
- **Требования:** учётная запись Microsoft (Office 365) и регистрация приложения Azure AD для получения OAuth-токена. Процесс настройки может быть чуть сложнее, чем с Gmail, но при наличии корпоративной почты Outlook это стоит сделать.
- **Полезность:** аналогична навыку Gmail — даёт возможность ассистенту напоминать о важных письмах, черновиках, событиях, не заходя вручную в почтовый клиент.

**Google Calendar** — навык интеграции с Google Календарём. Ассистент сможет узнавать ваши события и расписание, создавать новые встречи, напоминать о них. Например, можно спросить: «Что у меня запланировано на завтра?» — и ассистент, имея доступ к вашему календарю, ответит с перечислением встреч.
- **Требования:** OAuth-доступ к Google Calendar API (обычно настраивается вместе с Gmail либо отдельно через Google Cloud).
- **Полезность:** для пользователя, ценящего организацию времени, этот навык незаменим — OpenClaw может стать вашим персональным секретарём, который всегда скажет, что у вас на повестке.

**Email Summary (Сводка писем)** — полезный навык для автоматического обзора почты. Подключается к вашим почтовым ящикам (Gmail, Outlook, Yahoo и др.) и выбирает недавние письма, формируя краткие сводки.
- **Требования:** предоставить доступ к нужным почтовым аккаунтам (для Gmail/Outlook — OAuth, для других возможно IMAP-пароли).
- **Полезность:** особенно удобен пользователям, которые получают много писем. Ассистент, используя этот навык, может каждое утро выдавать краткий обзор новых писем, выделяя ключевые моменты — не нужно читать десятки сообщений полностью. Экономит время и снижает информационную перегрузку.

**Clippy (Microsoft 365)** — объединяет работу с Outlook-почтой и календарём через инструмент командной строки clippy.
- **Требования:** установка утилиты ms365/clippy и вход в Microsoft 365.
- **Полезность:** если вы активно пользуетесь экосистемой Microsoft (почта, календарь, OneDrive), этот навык обеспечивает комплексное управление ими через OpenClaw. Однако для начала можно обойтись отдельными навыками Outlook; clippy стоит подключить продвинутым пользователям для расширенных сценариев.

### Обмен сообщениями и уведомления

**Slack** — навык интеграции со Slack (корпоративный мессенджер). Позволяет ассистенту выполнять действия в Slack: отправлять сообщения в каналы, читать последние посты, отвечать на упоминания и т. д.
- **Требования:** Slack API-токен или установка Slack CLI. Скорее всего, понадобится создать Slack App с токеном пользователя или бота и поместить этот токен в настройки OpenClaw.
- **Полезность:** для пользователей, использующих Slack по работе, навык может экономить время. Например, можно попросить: «Напиши в канал #общий, что я сегодня работаю из дома», либо «Поиск по Slack: найди сообщение от босса с файлом отчёта». Ассистент выполнит это, избавив от необходимости переключаться в приложение Slack.

**Telegram (RemindMe)** — OpenClaw умеет работать и с Telegram. Прямая интеграция Telegram чаще настраивается через подключение бота, но есть и специальные навыки. Один из самых полезных — RemindMe, который присылает вам напоминания в Telegram. Вы можете задать ассистенту: «Напомни мне через 2 часа проверить запасы» — и навык `remindme` отправит вам личное сообщение-напоминание в Telegram в нужное время.
- **Требования:** привязанный Telegram-аккаунт/бот к OpenClaw (через официальный плагин или Bot API-токен).
- **Полезность:** отличное решение для пользователей, которые хотят получать уведомления от ассистента на телефон. Telegram очень популярен и у людей 40+, поэтому возможность получать заметки и предупреждения прямо в мессенджере делает OpenClaw более полезным в повседневной жизни.

**WhatsApp** — интеграция с WhatsApp также возможна. Существует несколько сторонних навыков (например, WhatsApp Styler, WhatsApp CLI (wacli) и др.), позволяющих ассистенту отправлять сообщения в WhatsApp или анализировать историю чатов.
- **Требования:** настройка сложнее, так как официальный API WhatsApp — платный (через Twilio или Cloud API). Некоторые навыки пытаются обходить это, эмулируя веб-клиент.
- **Полезность:** если критично получать ответы именно в WhatsApp, можно исследовать эти навыки, однако для начинающих пользователей рекомендуем более простые каналы (Telegram). Навыки WhatsApp можно подключить на более продвинутом этапе.

**Социальные сети** — OpenClaw может облегчить вашу активность в соцсетях. Например, навык Twitter (X) позволяет читать ленту Twitter, публиковать твиты и даже генерировать идеи для постов. Аналогично есть навыки для Mastodon (tootbot), для Facebook Messenger (messenger) и даже LinkedIn.
- **Требования:** как правило, нужны API-ключи соответствующей платформы или куки авторизации (для Twitter X сейчас часто используются cookie, учитывая ограничения API).
- **Полезность:** для пользователей, ведущих профессиональные страницы или просто желающих автоматизировать соцсети, эти навыки могут писать за вас черновики постов, планировать публикации (social-scheduler) и т. д. Однако если вы не являетесь активным пользователем соцсетей, на начальном этапе эти навыки можно не устанавливать.

### Органайзеры, заметки и задачи

**Apple Notes и Reminders** — если OpenClaw установлен на macOS, можно воспользоваться встроенными возможностями Apple. Есть навыки для приложений Заметки (Notes) и Напоминания (Reminders): они позволяют добавлять и искать заметки, создавать напоминания в стандартных приложениях Mac. Навык `apple-notes` работает через утилиту memo и AppleScript для управления Заметками (создание, редактирование, поиск). Apple Reminders аналогично работает через `remindctl` CLI для доступа к напоминаниям на Mac.
- **Требования:** Mac-компьютер (на Linux/Windows эти навыки не действуют) и установка вспомогательных CLI (memo, remindctl, настраиваются через Homebrew).
- **Полезность:** для пользователей экосистемы Apple — огромная. Вы сможете командовать ассистенту: «Добавь заметку в Apple Notes: купить лекарства» или «Создай напоминание на завтра 9:00 позвонить клиенту» — и OpenClaw добавит это в ваши приложения, как если бы вы сделали это вручную.

**Todoist** — популярный сервис для личных задач. Навык `todoist` интегрирует его с OpenClaw, позволяя создавать задачи, отмечать выполненные, просматривать список дел.
- **Требования:** API-токен Todoist (выдаётся в настройках аккаунта). Подключив его, вы даёте ассистенту доступ к вашим проектам и задачам.
- **Полезность:** можно просто сказать: «Добавь задачу: записаться к дантисту в понедельник» — и ассистент добавит задачу в ваш Todoist. Также можно спрашивать «Что у меня на сегодня по Todoist?» — и получать краткий отчёт о предстоящих делах. Это очень упрощает планирование дня.

**Trello** — если вы предпочитаете канбан-доски, навык `trello` поможет управлять досками Trello: создавать карточки, перемещать их между списками, читать комментарии.
- **Требования:** ключ и токен API Trello (получаются в профиле Trello).
- **Полезность:** если вы уже ведёте проекты в Trello, навык позволит взаимодействовать с ними голосом/чатом. Например: «Создай карточку в списке "В работе": Закупить офисную бумагу». Это скорее для продвинутых пользователей.

**Notion** — современный инструмент заметок и баз знаний. Навык `notion` позволяет создавать и редактировать страницы Notion через API.
- **Требования:** интеграция Notion API (создать интеграцию в Notion и получить секретный токен, подключить к нужным страницам).
- **Полезность:** если вы ведёте записи, базы данных или списки в Notion, ассистент сможет автоматизировать работу с ними. Например, по команде «Сохрани эту информацию в Notion страницу "Идеи"» — агент через навык добавит текст. Настройка API Notion — чуть более продвинутый шаг.

### Погода и новости

**Weather (Погода)** — простой, но полезный навык для получения текущей погоды и прогноза. Поддерживаются запросы типа «Какая погода будет завтра в Москве?» — ассистент выдаст температуру, осадки и т. д.
- **Требования:** нет, навык использует открытые источники (например, Open-Meteo или др.), API-ключ не нужен.
- **Полезность:** высокая, особенно для ежедневного использования. Вы можете по утрам спрашивать прогноз, не открывая отдельное приложение. Для пользователей, не слишком разбирающихся в гаджетах, это удобный способ получить сводку погоды голосом/чатом.

**News (Новости)** — единого официального навыка «Новости» нет, но есть несколько сторонних решений. Например, `google-news-api` — навык для парсинга ленты Google Новостей, может собирать свежие новости по темам. Другой подход — навыки под конкретные издания: например, `bbc-news` (получение последних новостей BBC через RSS).
- **Требования:** зачастую никакого, кроме интернет-доступа; иногда требуются API новостных сервисов.
- **Полезность:** для тех, кто хочет оставаться в курсе событий, ассистент с таким навыком может ежедневно формировать новостную сводку. Например: «Расскажи главные новости технологии за сегодня» — и на основе RSS-лент или API новостей навык предоставит краткий обзор. Это удобно, но на начальном этапе можно обойтись и просто веб-поиском.

### Работа с документами и файлами

**PDF Reader (Чтение PDF)** — OpenClaw можно научить читать и анализировать PDF-документы. Существует несколько навыков для этого. Один из популярных — навык `pdf` (и его обновлённая версия `pdf-2`), предоставляющий полный инструментарий для работы с PDF: извлечение текста, таблиц, разбиение на страницы и даже создание новых PDF.
- **Требования:** как правило, установка Python-библиотеки для работы с PDF (например, PyMuPDF или PDFMiner). Некоторые навыки могут скачивать нужную библиотеку автоматически, либо нужно вручную установить пакет (`pip install pymupdf`).
- **Полезность:** очень высокая для широкого круга пользователей. Вы можете попросить: «Прочти этот PDF-документ и сделай краткое резюме» — ассистент извлечёт текст и сгенерирует конспект. Особенно полезно для пользователей 40+, которые получают счета, отчёты, инструкции в PDF — не придётся вникать самим, AI-ассистент выдаёт суть. Навык PDF также удобен для поиска по документу: «Найди в этом PDF слово "гарантия"» — и ассистент найдёт нужное место. Рекомендуем подключить один из PDF-навыков в числе первых.

**Office-документы (Word, Excel)** — помимо PDF, есть навыки для форматов Microsoft Office. Например, `docx` — умение читать и редактировать Word-документы, `excel` — работа с таблицами Excel (XLSX).
- **Требования:** Python-библиотеки (python-docx, openpyxl и т. д.), которые навыки могут устанавливать.
- **Полезность:** если вы часто имеете дело с такими файлами, ассистент сможет по команде вытащить таблицу из Excel, посчитать что-то или создать черновик документа Word. Для базовых пользователей это, возможно, избыточно на старте, но знание о наличии таких навыков пригодится.

**Файловая система (File System)** — OpenClaw умеет и без специальных навыков выполнять некоторые операции с файлами (если вы разрешили доступ). Однако для расширенных функций существуют навыки вроде `clawdbot-filesystem` — поиск и массовая обработка файлов на диске.
- **Требования:** нет особых, работает с файловой системой сервера (нужны соответствующие права).
- **Полезность:** вы можете попросить ассистента найти файл по названию, просканировать папку на предмет определённых документов и даже выполнить резервное копирование важных данных. Например: «Найди все файлы Excel в папке Отчёты и посчитай их общий объём» — подобные задачи становятся тривиальными. Будьте осторожны: с таким доступом агент мощный, поэтому убедитесь, что включены только нужные навыки и настроена безопасность.

**Резервное копирование** — навык `simple-backup`: делает резервную копию «мозга» ассистента (рабочей папки и состояния) в указанное место.
- **Требования:** достаточное место на диске, настройка пути сохранения.
- **Полезность:** для пользователей, которые начинают активно использовать OpenClaw, полезно периодически сохранять всю память и файлы ассистента. Навык прост и может по расписанию (через cron, например) сохранять бэкап — это уже более продвинутый сценарий, но знать про него стоит.

### Системные утилиты

**System Info (Информация о системе)** — маленький, но удобный навык для мониторинга вашего сервера. Показывает загружённость процессора, использование памяти, время работы системы, состояние диска.
- **Требования:** никаких — доступ к системным метрикам есть у самого OpenClaw.
- **Полезность:** если OpenClaw установлен на сервере, особенно удалённом, вы можете спросить: «Как себя чувствует сервер?» — и ассистент (с этим навыком) выдаст, например: «ЦПУ: 30%, Память: 2 ГБ из 8 ГБ, Аптайм: 3 дня». Для пользователей 40+, не желающих лезть в консоль и вспоминать команды мониторинга, это простой способ следить за состоянием машины.

**VirusTotal Scan** — навык безопасности, позволяющий сканировать файлы и ссылки через VirusTotal. Если ассистент скачал какой-то файл по вашему запросу, вы можете попросить его проверить файл на вирусы. Навык отправит хеш или сам файл на VirusTotal и вернёт результат (какие антивирусы что обнаружили).
- **Требования:** бесплатный API-ключ VirusTotal (получается при регистрации на virustotal.com).
- **Полезность:** дополнительный уровень уверенности. Особенно если вы даёте ассистенту задачу загрузить что-то из интернета — стоит после проверить. Пользователю не нужно самому разбираться с антивирусами — OpenClaw с этим навыком сделает проверку за пару секунд.

**OpenClawGuard** — продвинутый навык для аудита безопасности самого OpenClaw. Он анализирует код запущенного ассистента, установленные навыки и плагины на наличие уязвимостей, подозрительного кода и пр. По сути, это сканер, который предупреждает, если какой-то навык может быть вредоносным.
- **Требования:** нет особых, кроме обновлённой базы сигнатур (возможно, навык сам её подтягивает).
- **Полезность:** рекомендован продвинутым пользователям, особенно если вы активно ставите сторонние навыки из интернета. В свете сообщений о том, что находили сотни вредоносных навыков в ClawHub, OpenClawGuard — очень полезное дополнение.

### Умный дом и устройства IoT

**Home Assistant** — сторонний навык для интеграции с популярной системой автоматизации Home Assistant. Если у вас развёрнут Home Assistant (он управляет всеми вашими умными лампочками, розетками, датчиками и т. д.), то подключив этот навык, вы получаете полный голосовой контроль над умным домом. Ассистент сможет выполнять команды типа: «Выключи свет в гостиной» или «Установи температуру термостата на 22 градуса».
- **Требования:** действующая инсталляция Home Assistant, long-lived access token (выдаётся в профиле HA) или URL вебхука.
- **Полезность:** крайне высока, если у вас много устройств — OpenClaw станет единым интерфейсом к ним. Однако настройка потребует некоторых усилий, поэтому это продвинутый шаг.

**Philips Hue** — навык для управления умным освещением Philips Hue (лампочки, светильники). Позволяет включать/выключать свет, менять яркость, цвет, активировать сцены.
- **Требования:** локальный Hue Bridge в вашей сети и подключённый API-ключ (получается путём нажатия кнопки на мостике и отправки запроса).
- **Полезность:** если у вас дома Hue, то команда «Сделай свет ярче на кухне» или «Включи вечернюю сцену» сработает через ассистента. Навык Hue входит в число предустановленных в OpenClaw, но заработает только после того, как вы выдадите доступ (ключ) и укажете IP вашего Hue Bridge.

**Sonos** — навык для аудиосистем Sonos (умные колонки). С его помощью можно командовать музыкальным воспроизведением: включать музыку, переключать треки, регулировать громкость в разных комнатах.
- **Требования:** наличие колонок Sonos в сети; обычно Sonos открыты для обнаружения, и навык сможет найти их, но возможно, понадобится указать IP.
- **Полезность:** любители музыки оценят возможность просто сказать «Включи джаз в гостиной» — и ассистент через Sonos начнёт воспроизведение. Навык особенно хорош для тех, кому сложно разбираться с мобильным приложением Sonos.

**Eight Sleep** — интересный официальный навык для владельцев умных матрасов Eight Sleep. Он позволяет регулировать температуру матраса, узнавать статистику сна.
- **Требования:** аккаунт Eight Sleep (подписка) и подключение по API.
- **Полезность:** нишевая — пригодится лишь тем, у кого есть такое устройство. Если у вас оно есть, ассистент сможет, например, нагреть постель к вашему отходу ко сну по команде «Подогрей кровать до 36°C».

**Tesla** — навык для владельцев электромобилей Tesla. OpenClaw сможет управлять некоторыми функциями авто: проверять уровень заряда, дистанционно запускать климат-контроль, открывать/закрывать замки.
- **Требования:** учётная запись Tesla; обычно интеграции требуют токен Tesla API (который нужно получить через сторонние утилиты, т.к. официально API не публичен).
- **Полезность:** для автовладельцев Tesla — очень высокая. Например, утром за завтраком можно спросить: «Сколько заряда в моей Tesla?» — ассистент ответит: «Заряд 80%, запас хода ~300 км». Или команда «Открой машину и включи обогрев» — и к вашему выходу из дома салон будет тёплым.

**Управление техникой** — кроме вышеупомянутых, есть навыки для множества устройств: Samsung Smart TV (через SmartThings), Amazon Alexa (навык `alexa-cli` для управления устройствами Alexa через неофициальный CLI), Netatmo (умные метеостанции), Sensibo (умные кондиционеры) и др. Если у вас есть умный гаджет, стоит поискать по названию на ClawHub — возможно, уже создан навык для него.
- **Требования:** разные для каждого (токены API, локальный доступ и пр.).
- **Полезность:** превращает OpenClaw в универсальный пульт для дома. Однако каждая такая интеграция — продвинутый сценарий. Рекомендуем сначала освоить более общие навыки, а потом по необходимости подключать интеграции устройств.

### Мультимедиа и развлечения

**Spotify** — несколько навыков позволяют ассистенту управлять воспроизведением Spotify. Например, `spotify-web-api` — использует официальный Web API Spotify для воспроизведения треков, получения вашего плейлиста, поиска музыки. Есть также `spotify-player (spogo)` — вариант, работающий через сторонний CLI spogo без официального API.
- **Требования:** для Web API — Spotify Developer Client ID/Secret и токен доступа (потребуется залогиниться через OAuth). Для CLI spogo — установка этого CLI и авторизация через него.
- **Полезность:** если вы любите слушать музыку на колонках или ПК, можно голосом включать любимые песни. «Поставь мой плейлист "80-е" на Spotify» — и мелодия зазвучит.

**YouTube и видео** — для любителей YouTube существует замечательный навык Transcript (или аналогичные), который получает расшифровку субтитров YouTube-видео и составляет краткое резюме. Это полезно, когда у вас нет времени смотреть длинный ролик — ассистент просматривает и пересказывает суть.
- **Требования:** навык `transcript` обычно использует сторонний сервис (например, TranscriptAPI). Хорошая новость: первая регистрация происходит автоматически на лету — агент сам получает API-ключ, вам не нужно ничего делать вручную, и даётся некоторое количество бесплатных расшифровок.
- **Полезность:** очень высокая. Вам прислали видеолекцию на 2 часа — вы говорите: «Дай конспект этого видео», агент расшифровывает речь и выделяет основные пункты. Для обучения, работы или просто экономии времени — бесценно.

**Видеоконференции** — существуют навыки для интеграции с Zoom, Google Meet и др., которые могут, например, записывать встречу, делать краткие заметки по итогам созвона и т. п. Это ещё очень новая область, и такие навыки требуют API-ключей соответствующих сервисов (Zoom API, Google Meet API). Если вы часто участвуете в онлайн-встречах, возможно, стоит исследовать эти возможности, но готовых отлаженных решений пока мало.

**Телефонные звонки и SMS** — интересная возможность: навыки вроде Pamela-call позволяют инициировать автоматические телефонные звонки с помощью AI-голоса. Например, OpenClaw может позвонить вам или кому-то и воспроизвести сгенерированное сообщение (синтезированным голосом). Или навык ElevenLabs `phone-reminder` отправит голосовое напоминание через звонок.
- **Требования:** Twilio или другой провайдер телефонии + аккаунт в сервисе синтеза речи (напр. ElevenLabs).
- **Полезность:** для очень узких сценариев (например, автоматические обзвоны). Для большинства пользователей это излишне, но впечатляет, на что способен ассистент. Относим к разряду экспериментов для энтузиастов.

### Помощь разработчикам и технические навыки

> *Если вы не являетесь программистом или администратором, вы можете пропустить этот раздел. Мы включаем его для полноты, т. к. среди официальных навыков OpenClaw есть и такие. Но для пользователя с базовыми навыками компьютера эти возможности вначале не нужны.*

**GitHub** — официальный навык `github` интегрирует GitHub CLI (gh) с ассистентом. Агент может создавать issue, комментировать pull requests, просматривать ваш репозиторий.
- **Требования:** установка GitHub CLI (gh) на сервере и залогиненный через него аккаунт GitHub.
- **Полезность:** для разработчиков, ведущих проекты на GitHub, ассистент может взять на себя часть работы. Например, по запросу «Создай issue в репозитории OpenClaw-Docs с названием "Ошибка в разделе Установка"» навык выполнит команду `gh issue create...`. Если вы не разработчик, этот навык ни к чему. Если да — весьма экономит время.

**Coding Assistant (AI Programmer)** — OpenClaw может помогать писать код, запуская встроенные AI-модели или внешние. Навык `coding-agent` запускает кодового ассистента (например, Claude Code, OpenAI Codex, Replit Ghostwriter) прямо внутри OpenClaw. По сути, при запросе на программирование, OpenClaw передаёт задачу специальной модели, заточенной под написание кода.
- **Требования:** доступ к модели: либо локально установленная (например, Codex CLI, Claude Code CLI), либо API-ключ (OpenAI).
- **Полезность:** для программистов — огромная (автогенерация функций, поиск ошибок, написание скриптов). Для непосвящённых — не актуально. Но знать, что OpenClaw способен даже писать код и скрипты по вашему описанию, — интересно само по себе.

**Command-Line Tools** — OpenClaw хорошо дружит с командной строкой. С помощью навыка `exec` (встроенный инструмент) он и так может запускать shell-команды. Но есть и навыки, которые упрощают работу со скриптами: например, `shell-scripting` — обучает агента писать shell-скрипты по лучшим практикам, или `sysadmin-toolbox` — подсказывает команды для администрирования серверов.
- **Требования:** никакого (кроме понимания, что вы делаете).
- **Полезность:** для ИТ-специалистов. Если вы не очень знакомы с терминалом — лучше воздержитесь от дачи ассистенту произвольных команд, чтобы не навредить системе. Но если вы продвинутый пользователь, OpenClaw может стать вашим голосовым DevOps-помощником.

**Другие навыки для кодеров** — существует множество сторонних скиллов: интеграция с Jira (`jira` — навык для задач разработки), управление Docker/Kubernetes, анализ логов (`clawdbot-logs`), даже генерация документации по коду. Полный их обзор выходит за рамки этого гайда. В категориях "Coding & IDE" и "DevOps" на ClawHub — сотни навыков.

### Прочие интересные навыки

**Picnic** — сторонний навык для заказа продуктов из супермаркета Picnic (служба доставки, популярна в некоторых странах Европы). Ассистент может искать товары, добавлять их в корзину, оформлять доставку.
- **Требования:** аккаунт в сервисе Picnic.
- **Полезность:** если вы пользуетесь доставкой еды/продуктов, это очень удобный навык — можно диктовать список покупок голосом. (Аналоги: навыки интеграции с Instacart, Amazon Fresh — тоже существуют.)

**Travel & Transport** — есть навыки для планирования путешествий: покупка авиабилетов, просмотр расписания поездов, заказ такси. Например, `uber-skill` (вызвать Uber через OpenClaw), `railway` (расписание поездов), `flight-status` (отслеживание статуса рейса).
- **Требования:** обычно API-ключи соответствующих сервисов (если открыты) или использование их публичных данных.
- **Полезность:** эпизодическая — может пригодиться, когда будете планировать поездку. Но на ежедневной основе, вероятно, не нужно держать их включёнными.

**Health & Fitness** — навыки для здоровья: Fitbit (интеграция с трекером для отчётов по шагам, сну), Oura (подключение Oura Ring), `calorie-tracker` и т. п.
- **Требования:** аккаунты в этих сервисах.
- **Полезность:** для следящих за здоровьем пользователей ассистент может стать «коучем», напоминая про активность или выводя статистику по запросу. Например: «Сколько я прошёл шагов за сегодня?» — если настроен навык Fitbit, ассистент ответит, не надо открывать приложение.

**Education & Reference** — огромное количество навыков обучающей природы. От простых (словарь, переводчик, генератор рецептов) до весьма продвинутых (навыки по психологии, коучингу, обучению языкам). Например, `japanese-tutor` — интерактивный репетитор японского языка, `crucial-conversations-coach` — тренер по непростым беседам.
- **Требования:** никаких, кроме интереса пользователя.
- **Полезность:** если вам близка тема, эти навыки могут дать новое переживание — разговор с AI-тренером или советчиком в определённой области. Для 40+ такие эксперименты могут быть увлекательны (например, практика иностранного языка с терпеливым AI-собеседником).

**Развлечения** — куда без них! OpenClaw можно научить играть с вами — есть навык `boggle` для игры в Боггл (слова в таблице), `xkcd` — чтобы получать случайный комикс XKCD, `trivia-quiz` — ассистент задаёт вам викторину. Даже навыки для генерации историй, анекдотов или гороскопов.
- **Требования:** нет.
- **Полезность:** поднимает настроение и позволяет освоиться с общением с ассистентом в ненапряжной форме. Попробуйте подключить парочку, когда настроите основные рабочие навыки.

## Резюме: что подключить в первую очередь

Для пользователя ~40 лет с базовыми цифровыми навыками, который только что развернул OpenClaw, рекомендуется следующий минимальный набор навыков для начала:

| Навык | Зачем |
|---|---|
| **Интернет-поиск** (Bing или DuckDuckGo) | Чтобы ассистент умел находить свежую информацию |
| **Электронная почта** (Gmail или Outlook) | Чтобы вы могли управлять почтой голосом |
| **Календарь** (Google Calendar) | Для получения расписания и создания событий |
| **Погода** | Быстрый прогноз без ключей API |
| **PDF/Документы** | Навык чтения PDF, чтобы ассистент мог изучать документы за вас |
| **Напоминания** (через Telegram или системные) | Чтобы OpenClaw мог напомнить вам о деле (например, навык `remind-me` для Telegram) |
| **Системная информация** | Полезно контролировать свой сервер (навык `system-info`) |
| **ClawHub** (менеджер навыков) | Навык `clawhub`, позволяющий искать и устанавливать новые навыки прямо из чата |

Подключив вышеупомянутые навыки и выполнив необходимые настройки (API-ключи и логины там, где нужно), вы получите персонального ассистента, готового помочь в повседневных делах. По мере роста уверенности можно экспериментировать с умным домом, музыкой, соцсетями и другими продвинутыми навыками.

OpenClaw — очень гибкая платформа, и сообщество постоянно добавляет новые навыки. Главное — понимать назначение каждого навыка и соблюдать осторожность с доступом. Этот гайд познакомил вас с текущими возможностями на 2026 год.
