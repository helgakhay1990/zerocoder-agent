# Второй сквозной прогон оркестра + замер «правки 1-й vs 2-й»

**Дата:** 2026-06-09 · **Ветка:** контентный конвейер (анонсы ИИ-Акселератора) · **Для:** сдача ДЗ, метрика зрелости системы.
**Связано:** ретро 1-го прогона — `docs/2026-06-08_akselerator-progon-retro_doc.md`.

## Зачем этот прогон

Ретро первого прогона (08.06) выявило: главное слабое место оркестра — **не генерация, а синхрон источников правды и правил**. `announce-writer` нарушал языковые правила (буква Ё, латиница CEO/контрибьютор), хотя в самоотчёте писал «вычищено». Правки делал оркестратор вручную.

После 1-го прогона внесли два фикса:
1. **`tone-of-voice.md`** — добавлен раздел языковых ограничений (без латиницы, без Ё, англицизмы → русские эквиваленты, override обращения брифом).
2. **`zerocoder-facts.md`** — Акселератор выведен из статуса «не продаётся».

**Гипотеза:** с правилами в источнике правды второй прогон потребует меньше ручных правок.

**Чистота замера:** во 2-м прогоне субагенту дан обычный бриф со ссылкой на базу знаний — язык-правила НЕ диктовались в промпте. Проверяли, подхватит ли он их сам из обновлённого `tone-of-voice.md`.

## Главный вывод: фикс источника правды и финальный гейт — не замена друг другу

Замер дал **более тонкую картину**, чем «стало меньше правок»:

1. **Смысловой слой ToV-фикс закрыл сам.** 2-й прогон без подсказки выдал «ИИ-мышление» и «архитектор цифровой команды» (точные подстановки вместо AI / AI-Native), не написал «AI», «оффер», «вебинар» в заголовках. Концептуальные правила из `tone-of-voice.md` субагент подхватил.
2. **Механический мусор всё равно течёт.** Буква Ё (10) и CEO/контрибьютор (6) остались в сырье несмотря на правило в источнике правды. **Один апдейт ToV нарушения у генератора не убирает.**
3. **Несущая защита — `editor`.** Финальный проход снял 15 из 16 нарушений (≈94%): Ё 10→0, англицизмы 6→1.
4. **`editor` не идеален.** Пропустил «вебинаров» (тот же класс, что «Q&A» в прошлый раз) — **финальный проход оркестратора всё ещё нужен**, но объём ручной работы упал с «вся чистка руками» до «1 правка».

**Метрика зрелости:** ручные правки оркестратора по языку: 1-й прогон — весь объём вручную; 2-й прогон — **1 правка** (остальное снял editor). Двухуровневая защита (правило в источнике правды + детерминированный гейт editor) работает; остаточный контроль за человеком сохраняется.

## Что это даёт для презентации ДЗ

Готовый измеримый сюжет «система учится»:
- было: субагент врёт в самоотчёте, чистим всё руками;
- стало: правило в источнике правды снимает смысловые нарушения у генератора, editor снимает механические, человеку остаётся 1 правка;
- честный остаток: editor не ловит 100%, поэтому финальный человеческий гейт — часть процесса, а не признак незрелости.
