# Агент-ассистент Zerocoder — установка и запуск

Многофункциональный агент-оркестратор для запусков онлайн-школы (вебинары, акции, интенсивы): концепции, лендинги, анонсы, презентации, гайды, ТЗ на монтаж. Работает как команда специализированных субагентов под единым оркестратором.

Полная карта системы — `docs/2026-05-29_orchestra-system-map_doc.md`.

---

## Что нужно заранее

- **Claude Code** установлен (CLI / десктоп / IDE-расширение).
- Доступ к моделям: часть оркестраторов работает на **Opus**, остальные — на **Sonnet**.
- (Опционально) ключи для MCP-серверов: **Brave Search** (поиск) и **Kinescope** (работа с записями эфиров).

## Установка за 4 шага

**1. Получить папку.**
```
git clone <адрес-приватного-репозитория>
cd Ai-homework
```

**2. Настроить MCP-серверы под себя.**
```
cp agent-assistant/.mcp.json.example agent-assistant/.mcp.json
```
Открыть `agent-assistant/.mcp.json` и подставить **свои** ключи (Brave, Kinescope). Файл в `.gitignore` — ваши ключи не утекут обратно в репозиторий.
Если поиск/Kinescope не нужны — соответствующие блоки можно удалить.

**3. Открыть папку в Claude Code из её корня.**
Субагенты (`.claude/agents/`) и скиллы (`agent-assistant/skills/`) подхватываются автоматически. После добавления новых субагентов Claude Code нужно перезапустить.

**4. Наполнить базу знаний под свою компанию.**
Папка `knowledge-base/` — источник правды для всех субагентов. Замените содержимое на своё:
- `zerocoder-facts.md` — продукты, тарифы, оффер, активная воронка
- `audience.md` — сегменты ЦА, боли, язык
- `speakers.md` — спикеры и их регалии
- `tone-of-voice.md` — тон голоса, регистры брендов

Без актуальной базы агенты будут переспрашивать, а не выдумывать — это by design.

## Как пользоваться

Пишите оркестратору задачу обычным языком — он сам выберет субагента по триггерам:

| Запрос | Кто подключится |
|---|---|
| «собери анонс на вебинар…» | announce-writer |
| «сделай структуру лендинга по референсу…» | landing-architect |
| «нужна концепция запуска…» | launch-strategist (+ подмастерья) |
| «гайд участника / презентация эфира…» | methodist |
| «что у конкурентов / аудит рынка…» | analyst-assistant (+ подмастерья) |
| «ТЗ на монтаж записи…» | video-edit-assistant (+ подмастерья) |

Под каждый запуск копируйте `agent-assistant/_state-template.md` → `_state.md` в папку проекта — это трекер стадий.

## Что НЕ входит в репозиторий (и почему)

`.gitignore` намеренно исключает:
- **`.mcp.json`** — содержит живые API-ключи. У каждого свой (см. шаг 2).
- **`settings.local.json`** — личные настройки Claude Code.
- **`.claude/agent-memory/`** — память агентов накапливается под конкретного пользователя.
- **`transcripts/`, `source/`** — записи и расшифровки эфиров (конфиденциально).
- **Тяжёлые медиа** (`*.mp4`, `*.pptx`, `*.pdf`, модели whisper) — раздаются отдельно при необходимости.

## Безопасность

- Никогда не коммитьте `.mcp.json` с реальными ключами.
- `knowledge-base/` может содержать внутренние данные компании (цены, контакты) — держите репозиторий **приватным**.
- Агенты по правилам не публикуют ничего во внешние сервисы и не отправляют письма/посты — только готовят черновики на проверку.
